Alpha Test

Ein Alpha-Test ist generell eine sehr frühe Phase in der Entwicklung eines Produktes. Nicht einmal alle Features sind fertiggestellt, geschweige denn gefunde Fehler behoben. Es erlaubt lediglich einen frühen Einblick in die Entstehung und erlaubt dem Entwickler Probleme auf den vielen verschiedenen Platformen früh zu erkennen. Ausserdem können Reaktionen und Wünsche der Benutzer unter Umständen berücksichtigt werden.

Daraus folgt, daß Benutzer in dieser Phase keine Erwartungen an das Produkt haben können oder sollten. Das gilt natürlich generell für alle Produkte unter freier Lizens aber ist in dieser Phase natürlich noch extremer.

Das Produkt kann sich im schlimmsten Falle auf eine Weise benehmen (den Akku leerziehen, das Handy überhitzen, die Grossmutter toasten), die nicht erwünscht sind. Es können keine Forderungen jeglicher Art in Konsequenz solcher Störungen abgeleitet werden. Im schlimmsten Fall kann ein Programm sogar Einfluss auf andere Software haben oder Hardware zerstören. Der Tester muss sich dieser Gefahr bewusst sein. Jeder Benutzer stimmt diesen Bedingungen zu. Möchte sie oder er dies nicht, muss er oder sie das Produkt deinstallieren.
